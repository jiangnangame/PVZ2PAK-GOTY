

PVZ2PAK is the best mod of PVZ1 which restores PVZ2 by JiangNanGame . We modify the files in PVZ1 including image, animate and program files to make it look like PVZ2.

We restarted this project in July 2022, and we named it PVZ2PAK:GOTY. As mensioned in the name, we uses the version of the Game of the Year, which different from the old version such as the Zombatar, Achievements system, Unlock All. We upload it to GitHub, hoping more people can experience our work.

Same as the old version, PVZ2PAK is free. We welcome everyone who follows The Changjiang River License to improve it after we stop updating and maintaining it.

We developed PVZ2PAK only for learning purposes. You must not use the program for commercial purposes, otherwise the legal issues should be resolved by yourself.
